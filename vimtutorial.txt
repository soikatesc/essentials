Modes:
-----
	1. Normal mode
		delete character -> x
		exit without writing ->  :q or :q!



	2. Insert mode
		press ->  i
		To back to normal mode -> esc
		save and quit -> :wq

		Navigating:
			h -> left to one word
			j -> up
			k -> down
			l -> right 

			w -> forward (one word)
			b -> begenning to each word
			e -> end of the word

			0 -> beginning of a paragraph
			$ -> end of a paragraph
